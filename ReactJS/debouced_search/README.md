# Problem Statement - Debouced Search Dropdown - like Google SearchBar
1. User should be shown Recipes based on input text using dummyrecipes API
2. Debounce the api call by delay say 300ms
3. On click outside of dropdown, it(suggestions dropdown) should close 
4. Cache the results for same api calls
5. On you press down arrow than the selected element from dropdown should update the text inside input
